## Atividade de Construção de Banco de Dados usando Java e Spring

Atividade se baseia na construção de banco de dados de consultas médicas.

As informações armazenadas para a consulta são:

- id
- Médico
  - id
  - nome
  - documento
  - especialidade
  - telefone
- prescrições
- exames
- prontuário
- data
